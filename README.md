# TIRESIAS_ENGINE - в разработке.
Этап разработки: build_index/

Некоммерческий прототип рекомендательной системы для конкретного сайта, работающий на ограниченных публичных данных (ежедневно/еженедельно обновляемые дампы баз данных без медиа). Цель — достичь качества рекомендаций, сравнимого с Pinterest, включая рекомендации по «доскам». 

Взаимодействие с целевым сайтом осуществляется через клиентскую надстройку (своё расширение или Tampermonkey): интерфейс (кнопки like/dislike, фильтры) встраиваются в DOM, а контент показывается модифицируя DOM с использованием ссылок к static (как на оригинальном сайте). Бэкенд (персональные профили, авторизация, онлайн-сервис) и надстройка не реализованы до тех пор, пока не будет доработано вычисление артефактов. 

Текущие модули "build_index/" — в рабочем состоянии, требуют доработки.
